# Deskripsi Program
Program Manajemen Nilai Mahasiswa dibuat untuk mengelola data nilai mahasiswa.
Program ini dapat menerima input data mahasiswa dan nilai, menghitung nilai
akhir, menentukan grade, serta menampilkan hasil pengolahan nilai tersebut.
Program ini dibuat sebagai tugas Ujian Akhir Semester (UAS).
# Repositori Isi
Repository ini berisi file dan folder yang digunakan dalam Program Manajemen
Nilai Mahasiswa. Di dalam repository ini terdapat file program utama yang
digunakan untuk mengelola dan menghitung nilai mahasiswa, serta file README
yang berisi penjelasan mengenai program, identitas pembuat, dan cara
penggunaan program. Repository ini juga dapat dilengkapi dengan folder
dokumentasi yang berisi tangkapan layar hasil menjalankan program sebagai
bukti bahwa program dapat berjalan dengan baik.
# Cara Pengguna
Untuk menggunakan Program Manajemen Nilai Mahasiswa, pengguna perlu
menjalankan program menggunakan compiler bahasa C. Setelah program
dijalankan, pengguna akan diminta untuk memasukkan data mahasiswa dan
nilai sesuai dengan petunjuk yang muncul di layar. Program kemudian akan
memproses data tersebut dan menampilkan hasil perhitungan nilai akhir
beserta grade mahasiswa.
# Catatan
Program ini dibuat untuk keperluan akademik sebagai tugas Ujian Akhir Semester
(UAS). Program masih dapat dikembangkan lebih lanjut sesuai kebutuhan.
